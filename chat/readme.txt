Chat. Decentralized. Stylish. Simple.

Usage: python chat.py [-h] [-room room]
Requirements: python v3+. No third-party modules are required.

Besides normal messages you can send commands. List of available commands:
1. /ban <nickname>. Block messages from peer with given nickname.
2. /unban <nickname>. Stop blocking messages from peer with given nickname.
3. /whisp <nickname> <message>. Send private message to peer with given nickname.
4. /join <ip address>:<port>. Join to room with given ip address.
5. /leave. Leave current room.
6. /nick <nickname>. Change current nickname to given one.
